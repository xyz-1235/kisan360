Defining & Calling Functions

A function is an independent, reusable block of code defined with the def keyword. It runs only when called.

def greet(name):
    print("Hello, " + name)

greet("Alice")   # Output: Hello, Alice
Types of Arguments

*args (Arbitrary Arguments) — function receives a tuple of arguments.

def my_func(*people):
    print(people[0])

my_func("Alice", "Bob")

Default Parameter — used when no argument is passed.

def country(name="Norway"):
    print("From: " + name)

country()  # From: Norway

List as Argument — any data type can be passed; treated same inside.

fruits = ["apple", "mango"]
my_func(fruits)

Return Statement — sends a value back to caller.

def add(a, b):
    return a + b

result = add(3, 5)  # 8
Recursion

A function that calls itself. Must have a base condition to stop.

def tri_recursion(k):
    if k > 0:
        result = k + tri_recursion(k - 1)
        print(result)
    else:
        result = 0
    return result

tri_recursion(6)  # 1 3 6 10 15 21
Base condition is k == 0; prevents infinite loops.

Lambda Functions

A small anonymous function — any number of arguments, but only one expression.

Syntax: lambda arguments : expression

x = lambda a: a + 10
print(x(5))   # 15

x = lambda a, b: a * b
print(x(5, 6))  # 30

With filter() & map()

nums = [1,2,3,4,5]
evens = list(filter(lambda x: x%2==0, nums))
# [2, 4]

doubled = list(map(lambda x: x*2, nums))
# [2, 4, 6, 8, 10]
filter() returns items where lambda → True; map() transforms every item.

Key function: open(filename, mode) — returns a file object.

Always call f.close() when done — unsaved changes may be lost.
Read Functions

read(size) — returns whole file or specified bytes.

f = open("demo.txt", "r")
print(f.read())     # full file
print(f.read(33))  # first 33 bytes

readline(size) — returns one line (or size bytes of it).

f.readline()   # 1st line
f.readline()   # 2nd line

readlines(hint) — returns list of all lines.

print(f.readlines())
# ['Hello!\n', 'Good Luck!']

Loop line by line

for line in f:
    print(line)
Write Functions

write(text) — writes string. With "a" appends; with "w" overwrites.

f = open("demo.txt", "a")
f.write("See you soon!")
f.close()

writelines(list) — writes list of strings (no auto newlines).

f = open("demo.txt", "a")
f.writelines(["Hi!", "Bye!"])
f.close()
Create & Delete Files/Folders

Create — use open() with "x", "a", or "w".

# Exclusive create (error if exists)
f = open("newfile.txt", "x")

# Create if not exists
f = open("newfile.txt", "w")

Delete — requires os module.

import os

# Delete file
os.remove("demo.txt")

# Check before delete
if os.path.exists("demo.txt"):
    os.remove("demo.txt")

# Delete empty folder
os.rmdir("myfolder")
os.rmdir() only works on empty folders.
Reading Config Files

Config files (.ini) group program settings into sections with key=value pairs. Use the configparser module.

from configparser import ConfigParser
cfg = ConfigParser()
cfg.read('config.ini')

cfg.sections()                         # list sections
cfg.get('installation', 'library')   # string value
cfg.getboolean('debug', 'log_errors') # bool value
cfg.getint('server', 'port')         # int value
